# U10｜測試策略與實戰

> 能為新方法撰寫單元測試與 Feature Test ｜ 120 min ｜ 前置依賴：U08

---

## 為什麼先教這個？

不寫測試的程式碼是「可能會動」，有測試的程式碼是「確定會動」。測試是修改程式碼後不會造成回歸錯誤的安全網——這是從「學習專案」走向「生產專案」的關鍵一步。

---

## 對應檔案

| 檔案 | 角色 |
|------|------|
| `tests/TestCase.php` | 測試基礎類別 |

| `tests/Feature/BookApiTest.php` | Feature Test（API 整合測試） |

## 核心觀念

### 測試金字塔

```
        /\
       /  \        E2E 測試（最少）
      /────\
     /      \      Feature Test（整合測試）
    /────────\
   /          \    Unit Test（最多、最快）
  /────────────\
```

- **Unit Test**：測試單一類別/方法，快速、隔離
- **Feature Test**：測試功能流程（HTTP 請求 → 回應），較慢但更貼近真實
- **E2E Test**：測試整個系統（瀏覽器自動化），最慢但最完整

### AAA 模式

```php
public function test_find_returns_book_when_exists()
{
    // Arrange（準備）
    $book = Book::factory()->create(['title' => 'Test Book']);

    // Act（執行）
    $result = $this->bookService->find($book->id);

    // Assert（驗證）
    $this->assertNotNull($result);
    $this->assertEquals('Test Book', $result->title);
}
```

### PHPUnit 基本斷言

```php
// 相等
$this->assertEquals($expected, $actual);
$this->assertSame($expected, $actual);  // 嚴格比較（型別 + 值）

// 真假
$this->assertTrue($condition);
$this->assertFalse($condition);

// Null
$this->assertNull($value);
$this->assertNotNull($value);

// 陣列
$this->assertCount(3, $array);
$this->assertContains($item, $array);
$this->assertArrayHasKey('key', $array);

// 例外
$this->expectException(NotFoundException::class);
```

### Laravel Feature Test

```php
use Illuminate\Foundation\Testing\RefreshDatabase;

class BookApiTest extends TestCase
{
    use RefreshDatabase;  // 每個測試前重建資料庫

    public function test_index_returns_all_books()
    {
        // Arrange
        Book::factory()->count(3)->create();

        // Act
        $response = $this->getJson('/api/books');

        // Assert
        $response->assertStatus(200)
            ->assertJsonCount(3, 'data');
    }

    public function test_store_creates_new_book()
    {
        $data = [
            'title' => 'New Book',
            'author' => 'Author',
            'isbn' => '978-1234567890',
            'stock' => 10,
        ];

        $response = $this->postJson('/api/books', $data);

        $response->assertStatus(201)
            ->assertJsonPath('data.title', 'New Book');

        $this->assertDatabaseHas('books', ['title' => 'New Book']);
    }
}
```

### 資料庫斷言

```php

$this->assertDatabaseMissing('books', ['title' => 'Deleted']);
$this->assertDatabaseCount('books', 3);

```

### Mock 依賴

```php
use Mockery;

public function test_service_with_mock()
{
    // 建立 Mock
    $mockRepo = Mockery::mock(BookRepository::class);
    $mockRepo->shouldReceive('find')
        ->with(1)
        ->once()
        ->andReturn(new Book(['id' => 1, 'title' => 'Mocked']));

    // 注入 Mock
    $service = new BookService($mockRepo);

    // 測試
    $result = $service->find(1);

    $this->assertEquals('Mocked', $result->title);
}
```

### Laravel 的 Mock 方法

```php
// Mock 並綁定到容器
$this->mock(BookService::class, function ($mock) {
    $mock->shouldReceive('find')
        ->with(1)
        ->andReturn(new Book(['title' => 'Mocked']));
});

// 或使用 partialMock
$this->partialMock(BookService::class, function ($mock) {
    $mock->shouldReceive('find')->andReturn(null);
});
```

### Factory 建立測試資料

```php
// 建立單筆
$book = Book::factory()->create();

// 建立多筆
$books = Book::factory()->count(3)->create();

// 自訂屬性
$book = Book::factory()->create([
    'title' => 'Custom Title',
    'stock' => 0,
]);

// 定義 Factory
// database/factories/BookFactory.php
class BookFactory extends Factory
{
    public function definition(): array
    {
        return [
            'title' => fake()->sentence(3),
            'author' => fake()->name(),
            'isbn' => fake()->isbn13(),
            'stock' => fake()->numberBetween(0, 100),
        ];
    }

    // 自訂狀態
    public function outOfStock(): self
    {
        return $this->state(['stock' => 0]);
    }
}

// 使用自訂狀態
$book = Book::factory()->outOfStock()->create();
```

## 動手做

### 必做

**1. 執行現有測試**

```bash
php artisan test

# 或使用 PHPUnit 直接執行
./vendor/bin/phpunit

# 執行特定測試
php artisan test --filter=BookServiceTest
```

**2. 建立 BookServiceTest**

```bash
php artisan make:test BookServiceTest --unit
```

編輯 `tests/Unit/BookServiceTest.php`：

```php
<?php

namespace Tests\Unit;

use App\Models\Book;
use App\Services\BookService;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Tests\TestCase;

class BookServiceTest extends TestCase
{
    use RefreshDatabase;

    private BookService $bookService;

    protected function setUp(): void
    {
        parent::setUp();
        $this->bookService = new BookService();
    }

    public function test_get_all_returns_all_books()
    {
        // Arrange
        Book::factory()->count(3)->create();

        // Act
        $result = $this->bookService->getAll();

        // Assert
        $this->assertCount(3, $result);
    }

    public function test_find_returns_book_when_exists()
    {
        // Arrange
        $book = Book::factory()->create(['title' => 'Test Book']);

        // Act
        $result = $this->bookService->find($book->id);

        // Assert
        $this->assertNotNull($result);
        $this->assertEquals('Test Book', $result->title);
    }

    public function test_find_returns_null_when_not_exists()
    {
        // Act
        $result = $this->bookService->find(99999);

        // Assert
        $this->assertNull($result);
    }

    public function test_create_stores_book_in_database()
    {
        // Arrange
        $data = [
            'title' => 'New Book',
            'author' => 'Author',
            'isbn' => '978-1234567890',
            'stock' => 10,
        ];

        // Act
        $book = $this->bookService->create($data);

        // Assert
        $this->assertDatabaseHas('books', ['title' => 'New Book']);
        $this->assertEquals('New Book', $book->title);
    }

    public function test_delete_removes_book_from_database()
    {
        // Arrange
        $book = Book::factory()->create();

        // Act
        $this->bookService->delete($book);

        // Assert
        $this->assertDatabaseMissing('books', ['id' => $book->id]);
    }
}
```

**3. 建立 BookApiTest**

```bash
php artisan make:test BookApiTest
```

編輯 `tests/Feature/BookApiTest.php`：

```php
<?php

namespace Tests\Feature;

use App\Models\Book;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Tests\TestCase;

class BookApiTest extends TestCase
{
    use RefreshDatabase;

    public function test_index_returns_all_books()
    {
        Book::factory()->count(3)->create();

        $response = $this->getJson('/api/books');

        $response->assertStatus(200)
            ->assertJsonCount(3, 'data');
    }

    public function test_show_returns_single_book()
    {
        $book = Book::factory()->create(['title' => 'Test Book']);

        $response = $this->getJson("/api/books/{$book->id}");

        $response->assertStatus(200)
            ->assertJsonPath('data.title', 'Test Book');
    }

    public function test_show_returns_404_when_not_found()
    {
        $response = $this->getJson('/api/books/99999');

        $response->assertStatus(404);
    }

    public function test_store_creates_book()
    {
        $data = [
            'title' => 'New Book',
            'author' => 'Author',
            'isbn' => '978-1234567890',
            'stock' => 10,
        ];

        $response = $this->postJson('/api/books', $data);

        $response->assertStatus(201)
            ->assertJsonPath('data.title', 'New Book');

        $this->assertDatabaseHas('books', ['title' => 'New Book']);
    }

    public function test_store_returns_422_when_validation_fails()
    {
        $response = $this->postJson('/api/books', []);

        $response->assertStatus(422)
            ->assertJsonValidationErrors(['title', 'author', 'isbn']);
    }

    public function test_destroy_removes_book()
    {
        $book = Book::factory()->create();

        $response = $this->deleteJson("/api/books/{$book->id}");

        $response->assertStatus(204);
        $this->assertDatabaseMissing('books', ['id' => $book->id]);
    }
}
```

## 踩坑提示

| 現象 | 原因 | 解法 |
|------|------|------|
| 測試時資料庫錯誤 | 沒有設定測試資料庫 | 在 `phpunit.xml` 設定 `DB_CONNECTION=sqlite` 和 `DB_DATABASE=:memory:` |
| 測試資料互相影響 | 沒有使用 `RefreshDatabase` | 在測試類別加上 `use RefreshDatabase` |
| Factory 找不到 | Model 沒有 `HasFactory` trait | 在 Model 加上 `use HasFactory` |
| Mock 沒生效 | Mock 在測試執行後才建立 | 確保 Mock 在呼叫前建立並綁定 |
| Feature Test 很慢 | 每次都真實存取 DB | 這是正常的，Feature Test 比 Unit Test 慢 |
